# CoTrip

CoTrip is a collaborative travel planning web app where users can create trip groups, invite members, collect candidate places, vote together, generate itineraries, and configure pre-trip reminders.

## Current Status

This repository is currently in the planning and implementation preparation stage.

## Planned Tech Stack

- Frontend: React + TypeScript + Vite
- Backend: AWS API Gateway + Python Lambda
- Database: Amazon RDS for PostgreSQL
- Authentication: Amazon Cognito
- Notifications: EventBridge Scheduler + Amazon SES
- Infrastructure: AWS SAM
